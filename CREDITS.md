# Crédits — assets tiers

Tous les assets 3D du Labo proviennent de **[Poly Haven](https://polyhaven.com)** et sont en
**CC0** (domaine public — aucune attribution obligatoire, usage commercial libre). Listés ici par
courtoisie.

- HDRI : **Forest Slope** → `public/hdris/forest_slope_1k.hdr`
- Texture sol : **Forest Floor** → `public/textures/forest_floor_*`
- Arbre : **Island Tree 02** → décimé en 2 LODs (gltf-transform, géométrie simplifiée + textures WebP)
  → `public/models/trees/island_lod0.glb`, `island_lod1.glb`
- Rochers / tronc (scans) : **Boulder 01**, **Moon Rock 01/02**, **Dead Tree Trunk** → `public/models/`

Aucune attribution n'est légalement requise (CC0).
